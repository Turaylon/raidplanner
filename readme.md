## RaidPlanner for Wildstar

RaidPlanner for the guild [TheTheory.it](http://www.thetheory.it) for [Wildstar](wildstar-online.com) game.

## Tecnology

RaidPlanner is based on:

- Laravel  PHP Framework 4.*
- Twitter Bootstrap 3.*
- jQuery 2.*
